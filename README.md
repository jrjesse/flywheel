# Sales Automation System

Sistema completo de ingestão e automação de funil comercial construído sobre React 18, Next.js 15, TailwindCSS e Spring Boot 3.4. O sistema usa Kafka para mensageria assíncrona.

## Features

- **Ingestão Webhook**: Aceita requisições JSON de diversas plataformas e classifica em leads.
- **Kafka Producer/Consumer**: Arquitetura Event-Driven.
- **SLA Tracker Automático**: Mede o tempo de resposta e aplica tag de Atraso automaticamente.

💡 **Diferencial (Kanban Module)**
"Implementei um painel Kanban dinâmico para gestão de leads com suporte a drag-and-drop, atualização de status em tempo real e histórico completo de interações, permitindo maior visibilidade operacional e controle de SLA."
